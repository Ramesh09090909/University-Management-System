The University Management System (UMS) is a desktop application built in Java (Swing + JDBC) to manage various university operations such as student details, teacher details, course management, examination records, and fee structure.

It provides an easy-to-use interface for administrators to insert, update, and view university data, reducing manual work and improving efficiency.

Features

Student Management

Add, update, and view student records

Store academic details 

Maintain contact and personal details

Teacher Management

Add, update, and view teacher information

Track subject and department details

Manage leave requests

Examination Module

Record and view examination results

Generate reports using database queries

Fee Structure

Leave Management

Students and teachers can apply for leave

Admin can view leave requests and status
